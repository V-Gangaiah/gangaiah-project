# 💡 Light ON/OFF Project

## 📌 Description
This is a simple web project that demonstrates how to turn a light bulb ON and OFF using JavaScript.

## 🎯 Objective
To understand basic DOM manipulation using JavaScript.

## 🚀 Features
- Turn light ON
- Turn light OFF
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
- When the user clicks the "Turn ON" button, JavaScript changes the image source to a glowing bulb.
- When the user clicks the "Turn OFF" button, JavaScript changes the image to a normal bulb.

## 📂 Files Included
- index.html → Structure of the webpage
- style.css → Styling and design
- script.js → Functionality using JavaScript

## ▶️ How to Run
1. Download all files
2. Open index.html in any browser
